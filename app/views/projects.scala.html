Diaspora

Diaspora is a free personal web server that implements a distributed social networking service. You'll find more information on diasporafoundation.org or wiki.diasporafoundation.org.

Diaspora Pods

A pod is a server running the diaspora* software and connected to the diaspora* network. "Pod" is a metaphor referring to pods on plants which contain seeds, in the way that a server contains a number of user accounts.

sechat.org

Sechat* runs the latest development version of Diaspora.

joindiaspora.com

Joindiaspora* runs the latest stable version of Diaspora.
